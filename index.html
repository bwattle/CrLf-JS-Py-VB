Concatenation in 3 languages.

An experiment with "carriage return" and "line feed" in JavaScript, Python and Visual Basic

The repository for this site is at: https://github.com/bwattle/CrLf-JS-Py-VB

Links to languages are below:

html - The html version loads text boxes when the file is opened. Hit F12 to see the code in the inspector. Apostrophe's can be handled by double quotes or a backslash.

Python py file - The Python (.py) file can be run in Visual Studio or run directly with Idle. Apostrophe's can be handled by double quotes or a backslash.

Python code - The Python code can be read at this link.

Visual Basic exe - the VB exe is at this link and the code can be found in the CrLf form folder. Apostrophe's present no problem in languages which use "double quotes".

Visual Basic code - the VB code can be read at this link.

 

A zip of the Visual Studio solution folder containing all 3 languages can be downloaded here.
